# CS4DS Course Materials

This repository contains the CS4DS materials taught by Tzu-You Lin in NTU ECON Programming (`ECON1024`) on a bi-weekly schedule.  
It is for this CS4DS course materials only.

## Course Focus

- Programming fundamentals for data analysis
- Core computational thinking for empirical economics
- Practical workflows for reproducible research

## Repository Structure

- `annoucement/` - course announcements and administrative updates
- `lecture1/` to `lecture6/` - lecture materials, examples, and exercises

## Lecture Subjects

- `lecture1` - The Shell
- `lecture2` - Programming Languages and Tools in Data Science/Economics
- `lecture3` - Introduction to Markdown and LaTeX
- `lecture4` - Git and GitHub
- `lecture5` - Web Scraping and APIs in Python
- `lecture6` - Introduction to Databases and Basic SQL Syntax

## Learning Outcomes

By the end of the course, students should be able to:

- Read and write basic programs for data tasks
- Use programming to support economic modeling and empirical analysis
- Organize and document code for reproducible projects

## How To Use This Repository

1. Start with the earliest lecture folder and proceed in order.
2. Read lecture notes before running examples.
3. Practice by modifying sample code and completing exercises.
4. Keep your own notes and scripts in a separate folder or branch.
